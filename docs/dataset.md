# Dataset Documentation

- Source: Craigslist Car Listings (Kaggle)
- Dataset Source: https://www.kaggle.com/....
Not stored in the repo due to size limits.

- Downloaded: 2025-11-18
- File format: CSV
- Columns:
  - id - entry ID
  - url - listing URL
  - region - craigslist region
  - region_url - region URL
  - price - entry price
  - year - entry year
  - manufacturer - manfacturer of vehicle
  - model - model of vehicle
  - condition - condition of vehicle
  - fuel - fuel type
  - odometer - miles traveled by vehicle
  - title_status - title status of vehicle
  - transmission - transmission of vehicle
  - VIN - vehicle identification number
  - drive - type of drive
  - size - size of vehicle
  - type - generic type of vehicle
  - paint_color - color of vehicle
  - image_url - image URL
  - description - listed description of vehicle
  - county - (useless column in dataset, 'left in by mistake')
  - state - state of listing
  - lat - latitude of listing
  - long - longitude of listing
  - postin_date - date of posting
